
# 🎓 Student ID Card — IIT

A clean, minimal **Student ID Card** built with pure HTML and CSS. Designed for Indian Institute of Technology (IIT) students, this single-page form renders a styled card UI that collects essential student details.

---

## 📸 Preview

A bordered card centered on the page with the IIT branding, blue-themed styling, and form fields for student information.

---

## 📁 Project Structure

```
student-id-card/
└── idcard.html       # Single-file project (HTML + CSS)
```

---

## ✨ Features

- **Single-file** — no dependencies, no build tools required
- **IIT-branded** card design with blue color scheme
- Student information fields:
  - Full Name
  - Enrollment ID
  - Branch
  - Date of Birth
  - Year
  - Gender (dropdown)
- Responsive, centered card layout
- Clean form UI with consistent styling

---

## 🚀 Getting Started

No installation needed. Just open the file in any browser:

```bash
# Clone the repo
git clone https://github.com/your-username/student-id-card.git

# Open in browser
open idcard.html
# or on Windows
start idcard.html
# or on Linux
xdg-open idcard.html
```
